## BITRIX DOCKER (ALPHA)
Контейнеры для развертывания\установки сайта на битрикс

## В сборке
- PHP 7.4.33 (cli) (gd, xdebug 3.1.6, )
- mysql 5.7

## Начало работы
- Склонируйте репозиторий bitrix-docker
```
git clone https://github.com/alekslesik/bitrix-docker
```
